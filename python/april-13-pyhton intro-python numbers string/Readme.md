## 🤖 AI vs Gen AI vs Agentic AI

### 🔹 Artificial Intelligence (AI)

- Developers train models using:
  - Data (Excel, images, videos, audio)
- Used for prediction and automation

---

### 🔹 Generative AI

- Models trained by companies (OpenAI, Google, Meta, etc.)
- Generates:
  - Text
  - Code
  - Images
  - Audio
  - Video

---

### 🔹 Agentic AI

- Autonomous agents connected to:
  - LLM models
  - Web tools
- Capabilities:
  - Write code
  - Debug
  - Compile
  - Install frameworks
  - Build complete applications

📌 Impact:

- 10 developers’ work → 3–4 developers using Agentic AI
- Cost saving and automation
- Increase in layoffs but future demand remains

## 🐍 Python Fundamentals

### Why Python?

- Easy compared to Java, C, C++
- Less code (Java: ~20 lines vs Python: ~4 lines)
- Object-Oriented Programming
- Platform independent (Windows, Mac, Linux)

### Usage Areas:

- Data Science
- Machine Learning
- AI & Gen AI
- Cloud
- Scripting

### Limitations:

- Not suitable for mobile app development (Android)

---

## 🧠 Python Facts

- Created in 1989
- Popular after 2017 (AI boom)
- Name inspired by **Monty Python’s Flying Circus**
- Uses Python Virtual Machine (PVM)

---

## ⚙️ Interpreter vs Compiler

### Interpreter

- Executes code line by line
- Example: Python

### Compiler

- Executes entire code at once

---

## 🔢 Data Types

### 1. Numbers (Numerical Data)

- Used for calculations
- Supports operations like BODMAS

### 2. Strings (Text Data)

- `' '` → Single quotes
- `" "` → Double quotes
- `''' '''` → Multi-line text / comments

📌 `_` symbol refers to **previous output in Python**

---

## 🧮 BODMAS Rule

Order of execution:

1. Brackets
2. Orders (powers)
3. Division
4. Multiplication
5. Addition
6. Subtraction
